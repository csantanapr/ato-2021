## Demo for All Things Open 2021

## Prerequisites

Check [OPS.md](OPS.md)

## Livereload

```
npm run dev
```

## Debug

```
npm run debug
```

## Bonus: Livereload with Serverless

Check [OPS.md](OPS.md) on to install knative

To develop
```
npm run dev-knative
```

To get url of knative service
```
npm run dev-knative-url
```

To debug
```
npm run dev-knative-debug
```
